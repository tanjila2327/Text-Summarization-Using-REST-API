# Text-Summarization-Using-REST-API

## Overview

Text Summarization Using REST API is a Flask-based web application that provides secure RESTful APIs for managing and summarizing text data. The project includes JWT authentication, CRUD operations, MySQL database integration, and API documentation using Swagger UI.

---

## Features

* RESTful API development using Flask
* JWT Authentication & Authorization
* CRUD Operations (Create, Read, Update, Delete)
* MySQL Database Integration
* API Documentation using Swagger UI
* Secure API endpoints
* Simple and user-friendly web interface

## Installation

### 1. Clone the Repository

```bash id="4gcfgr"
git clone https://github.com/your-username/Text-Summarization-REST-API.git
```

### 2. Navigate to Project Folder

```bash id="x0kxmu"
cd Text-Summarization-REST-API
```

### 3. Create Virtual Environment

```bash id="pj7x2v"
python -m venv venv
```

### 4. Activate Virtual Environment

#### Windows

```bash id="e7k4bp"
venv\Scripts\activate
```

#### Mac/Linux

```bash id="h75i4o"
source venv/bin/activate
```

### 5. Install Dependencies

```bash id="c45jow"
pip install -r requirements.txt
```

### 6. Run the Application

```bash id="wkrvrl"
python app.py
```
